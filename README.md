<summary><h2>MiniCalc</h2></summary>

It's like a calculator on your own cli 🤩

command: **minicalc** ->

- add num1 num2
- sub num1 num2
- mul num1 num2
- div num1 num2
- mod num1 num2

Example - minicalc add 11 12
